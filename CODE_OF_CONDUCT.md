# Code of Conduct

This project follows the Contributor Covenant code of conduct. By participating, you agree to abide by its terms.

Expected behavior
- Be welcoming, respectful, and collaborative.
- Provide constructive feedback and assume good faith.

Unacceptable behavior
- Harassment, hate speech, or abusive language will not be tolerated.

Reporting
- If you witness unacceptable behavior, please contact the maintainers or open an issue marked private.
